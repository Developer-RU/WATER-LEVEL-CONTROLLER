# Support

## Usage Questions

For setup and operation questions, open a GitHub issue with:
- Hardware board model.
- Relay module type (2CH or 4CH).
- Wiring summary.
- Current settings.
- Serial logs and screenshots.

## Bug Reports

Please include:
- Expected behavior.
- Actual behavior.
- Steps to reproduce.
- Firmware version.
- Relevant logs from serial monitor.

## Feature Requests

When requesting new capabilities, include:
- Practical use case.
- Why current implementation is insufficient.
- Proposed behavior.
