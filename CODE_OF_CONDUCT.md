# Code Of Conduct

## Our Commitment

We are committed to a respectful and professional collaboration environment.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical facts and reproducible evidence.
- Provide clear context when reporting issues.
- Help maintain safety and reliability standards for the project.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Deliberate misinformation in technical discussions.
- Unsafe recommendations that ignore documented safety constraints.

## Enforcement

Project maintainers may moderate discussions, issues, and pull requests to preserve a constructive and safe environment.

## Reporting

If you experience unacceptable behavior, report it to the repository owner.
